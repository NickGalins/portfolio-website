# Image Assets

Place project images here. For blog-specific images, see [blog/README.md](blog/README.md).

## Naming Convention

Use the project ID as a prefix:
- `everland-trust-safety-thumb.png` (thumbnail for homepage card)
- `everland-trust-safety-flow.png` (content image within project)
- `aws-ai-agents-thumb.png`

## Image Specifications

| Type | Dimensions | Format | Notes |
|------|------------|--------|-------|
| Homepage card thumbnails | 600x400px (3:2) | PNG/JPG | Used on homepage project grid |
| Content images | Max 1200px wide | PNG/JPG | Used within project pages |
| Author photos | 256x256px (1:1) | JPG | Circular crop in templates |

## Optimization

- Compress with [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/)
- Target file size: Under 200KB for large images, under 100KB for thumbnails
- Use PNG for screenshots/graphics, JPG for photos
