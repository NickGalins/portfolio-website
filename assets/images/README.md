# Image Assets

Place your project images here.

## Naming Convention

Use the project ID as a prefix:
- `everland-trust-safety-thumb.png` (thumbnail)
- `everland-trust-safety-flow.png` (content image)
- `aws-ai-agents-thumb.png`

## Recommendations

- Thumbnails: 800x450px (16:9 ratio)
- Content images: Max 1200px wide
- Format: PNG for screenshots, JPG for photos
- Optimize with TinyPNG.com before adding
