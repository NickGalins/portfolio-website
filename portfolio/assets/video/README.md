# Video Assets

Place your trailer video here as `trailer.mp4`.

## Recommendations

- Format: MP4 (H.264 codec)
- Duration: 15-30 seconds
- Resolution: 1920x1080 or 1280x720
- File size: Under 10MB for fast loading
- No audio needed (video is muted)
